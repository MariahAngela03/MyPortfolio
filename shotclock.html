24 - SECOND SHOTCLOCK CIRCUIT

The "24 Seconds Shot Clock" is a digital countdown timer system designed and implemented by Computer Engineering students at the University of Batangas – Lipa Campus. The project emulates a standard basketball shot clock by counting down from 24 seconds and displaying the remaining time using dual 7-segment displays. It was developed using a combination of digital integrated circuits such as the 74192 counter, 7448 BCD to 7-segment decoder, and a 555 timer configured to generate clock pulses. The entire system was assembled on a printed circuit board (PCB), which was designed using Eagle CAD software and physically implemented through etching, drilling, and soldering techniques. This project serves as a practical application of logic and circuit design principles covered in the CPE314 course.

The main challenges faced during the project include:

• Creating a countdown timer that reliably displays values on dual 7-segment displays.

• Synchronizing multiple ICs (e.g., 74192, 7448, 555 timer) to function cohesively within the circuit.

• Implementing control logic for the set, reset, and pause buttons.

• Designing a system that activates a buzzer at exactly zero seconds.

• Ensuring a modular, organized, and functional PCB layout, which required careful partitioning of subsystems.

• Dealing with real-world circuit issues like signal integrity and component coordination during physical implementation (soldering, etching, and testing).

To address these challenges, the students applied a systematic engineering approach, which included:

• Subsystem Partitioning: The entire system was divided into five manageable modules: countdown circuit, 555 timer oscillator, reset circuit, pause functionality, and buzzer/display control.

• Use of 74192 IC: This IC handled the countdown logic, paired with a 7448 BCD decoder for 7-segment display output.

• 555 Timer: Configured in astable mode to generate the clock signal required for countdown pulses.

• Buzzer Activation: A NOT gate (7404) and OR gate (7432) combination was used to activate a buzzer precisely when the countdown reached zero.

• PCB Design and Fabrication: Eagle CAD was used to create efficient PCB layouts. The boards were manufactured using standard processes such as etching, drilling, and soldering.

Key Features

  • 24-Second Countdown Timer
  • Reset Functionality
  • Set Function (Start)
  • Pause Capability
  • Buzzer Activation at Zero
  • User-Friendly Interface
