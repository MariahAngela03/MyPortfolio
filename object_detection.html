Project Details

Machine Learning
December 2024

Visionbot

The project is a local web-based application designed to perform real-time object detection and generate descriptive text based on identified items. It offers two main modes: live camera detection and image upload detection, providing users with flexibility in how they interact with the system. The application uses a custom-trained YOLOv8 model to accurately detect and label objects within images or live video feeds. A descriptive summary of each detected object is generated to provide contextual information, enhancing user understanding. Developed using Python and built with Streamlit, the system incorporates tools such as OpenCV for video and image handling, Roboflow for dataset annotation, and Google Colab for model training. This project serves as a functional and educational platform for exploring object detection and contextual analysis in visual data.

Challenges Encountered:

• Lighting & Camera Quality: Poor lighting and camera quality reduced detection accuracy.

• Labeling Errors: Some objects were incorrectly named or described.

• Object Closeness & Obstruction: Close or blocked objects were often misidentified or not detected.
