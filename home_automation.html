Home Automation System

The Home Automation System is an embedded systems project developed in August 2024 that integrates a web-based platform with various household appliances. The system allows users to remotely control appliances such as lights, fans, and other electronic devices through an intuitive website interface. It also offers manual control options, providing flexibility based on user preference or connectivity status. Designed with multi-user access in mind, the system logs user actions, enabling the admin to monitor which registered user toggled specific appliances, along with accurate timestamps.

The project aimed to address several challenges:

• Device Synchronization: Ensuring accurate, real-time reflection of appliance states on the website, especially during simultaneous remote and manual control.

• Hardware Integration: Integrating microcontrollers with various types of appliances while maintaining safety and efficiency.

• Data Logging Accuracy: Developing a reliable method to timestamp and log user actions without delay or duplication.
